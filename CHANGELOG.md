# Changelog

## [0.1.0] - 2026-06-16

### Added

- Visual HTML editing panel in VS Code
- Click-to-select elements on page
- Style editing: typography, layout, background, spacing, border, shadow
- Image controls: size, object-fit, border radius
- Parent/child element navigation (⬆ ⬇)
- Language toggle (中文 / English)
- Manual save via Cmd+S
- Undo / Redo
- Draggable, dockable panel
